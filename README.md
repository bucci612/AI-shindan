# AI-shindan

初心者向けのAI副業診断Webアプリです。

## 🚀 概要

いくつかの質問に答えることで、
あなたに合ったAI副業タイプを診断します。

## 🎯 できること

- 質問に回答
- タイプ別結果表示
- シンプルなUI
- スマホ対応
